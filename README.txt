== Description ==

Pilvi-embed is a plugin that enables easy embedding of Pilvi platform's features into Wordpress.

Currently supports embedding of
- Product Cards
- Product Card Groups

See Pilvi Developer site for more information. 
See http://developer.pilvi.com/docs/embeds

 
== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `pilvi-embed` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Choose version and fill up other settings from wordpress dashboards pilvi-embed menu.
4. Go to pages/posts tiny mce editor and choose shortcode from pilvi-embed button init.
   (More shortcode variants can be copypasted from available shortcodes section in pilvi-embed menu)
5. Add id to shortcodes id atribute.
6. Publish or update page.
